## About

The tutorials demonstrate how to use MONAI Label as an AI-assisted interactive labeling tool for medical image segmentation. This Hello World example covers:

- Installing and configuring MONAI Label
- Launching the MONAI Label server with the Radiology app
- Connecting 3D Slicer as the client viewer
- Running automated spleen segmentation with a pre-trained model
- Editing annotations and submitting labels
- Active learning workflow for continuous model improvement
